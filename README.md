# 🤖 Customer Segmentation using K-Means Clustering
**Tool:** Python | Scikit-learn | Pandas | Matplotlib  
**Domain:** Machine Learning | Unsupervised Learning  
**Author:** Roshani Mane | MBA – Business Analytics, DPU Pune

---

## 📌 Project Overview
This project applies K-Means Clustering on mall customer data to segment 
200 customers into 5 distinct behavioral groups based on their 
Annual Income and Spending Score.

---

## 🎯 Objective
To help businesses identify distinct customer segments and design 
targeted marketing strategies for each group.

---

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| Pandas | Data loading & manipulation |
| Matplotlib | Data visualization |
| Scikit-learn | K-Means clustering model |
| Google Colab | Development environment |

---

## 📊 Project Steps
1. **Data Collection** – Downloaded Mall Customers dataset (200 records)
2. **EDA** – Explored data, checked missing values, visualized distributions
3. **Elbow Method** – Determined optimal number of clusters (k=5)
4. **K-Means Clustering** – Segmented customers into 5 groups
5. **Visualization** – Scatter plot of all customer segments
6. **Business Insights** – Actionable recommendations per segment

---

## 👥 Customer Segments Found

| Segment | Avg Age | Avg Income | Avg Spending | Count |
|---------|---------|------------|--------------|-------|
| 🔴 Careful Spenders | 42.7 | $55k | 49.5 | 81 |
| 🔵 Target Customers | 32.7 | $86k | 82.1 | 39 |
| 🟢 Budget Customers | 25.3 | $25k | 79.4 | 22 |
| 🟠 Impulsive Buyers | 41.1 | $88k | 17.1 | 35 |
| 🟣 Average Customers | 45.2 | $26k | 20.9 | 23 |

---

## 💡 Key Business Insights
- 🎯 **Target Customers** (Blue) = Highest priority — high income & high spending
- 💡 **Impulsive Buyers** (Orange) = Biggest opportunity — earn $88k but spend only 17/100
- 🛍️ **Budget Customers** (Green) = Young shoppers (avg 25) who love spending despite low income
- 📢 **Careful Spenders** (Red) = Largest group (81 customers) — need value-for-money campaigns
