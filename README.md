# 📊 Sales & Customer Intelligence Dashboard
**Tool:** Power BI | DAX | Excel  
**Domain:** Business Analytics | Data Visualization  
**Author:** Roshani Mane | MBA – Business Analytics, DPU Pune

---

## 📌 Project Overview
This project is an interactive, multi-page Power BI dashboard built to analyze 
sales performance, customer behavior, profit margins, and return rates across 
product categories — enabling data-driven business decisions.

---

## 🎯 Objective
To transform raw sales data into actionable insights through visual storytelling, 
helping stakeholders quickly identify trends, problem areas, and growth opportunities.

---

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| Power BI | Dashboard design & visualization |
| DAX | Custom KPI measures & calculations |
| Excel | Data source & preprocessing |

---

## 📂 Dashboard Pages
- **Page 1 – Sales Overview:** Revenue trends, YoY growth, monthly performance
- **Page 2 – Profitability Analysis:** Profit margins by category, loss detection
- **Page 3 – Customer Behavior:** Return rates, customer lifetime value, segments

---

## 💡 Key Insights
- 📉 Identified **~18% profit leakage** in one product segment through trend analysis
- 📈 Built **10+ DAX measures** including YoY Growth, CLV, and Return Rate
- 🧩 Designed a **Star Schema** data model for optimized query performance
- 🎯 Delivered findings as a business case recommendation to course faculty
